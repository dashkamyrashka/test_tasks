ЗАДАНИЕ 1.

Даны 3 массива произвольной длины с произвольными данными. Создать новый массив, который будет содержать суммы элементов соответствующих массивов.

Во входные данные вводим 3 числовых массива, через пробел. В поле ввода действуют 2 подсказки: 	1) Видна до введения данных;
																								2) Видна при наведнеии на поле ввода.
Нажимаем кнопку "Сложить". 
Результат поэлементного сложения 3-х массивов появляется в поле под кнопкой, ответ записан через пробел.

ЗАДАНИЕ 2.

Написать функцию для перемешивания элементов массива. Т.е. при каждом вызове функции, сортировка массива должна меняться.

Во входных данныч вводим числовые элементы массива, через пробел.
Нажимаем кнопку "Перемешать".
И в результат получаем новый массив с теми же элементами, только в другом порядке.


ЗАДАНИЕ 3.

Написать модуль, который будет считать заданные математические выражения.

Во входные данные вводим выражение, где каждый элемент выражения записывается через пробел. Например: ( 2 + 1 ) + 3 * 2
Нажимаем кнопку "Ответ".
И в поле под кнопкой выводится ответ.
